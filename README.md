# First project Создание игрального кубика. Работу выполнили Петрова София и Усманов Кирилл. ИР 23-4
При нажатии кнопки "Бросить кубик" выводится случайное число от 1 до 6 и используется анимация для картинки. Для кнопки использовали "Button", для изображения - "ImageView"
